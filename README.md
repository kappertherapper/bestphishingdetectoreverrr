Best Phishing Extension Everrrrrr

A lightweight browser extension that detects and blocks phishing websites in real-time.

Features

Real-time phishing detection: Analyzes URLs and page content for suspicious patterns.
Instant alerts: Warns users when they visit a potentially malicious site.
Educational insights: Explains why a site is flagged, helping users recognize future threats.
Automatic blocking: Prevents access to known phishing sites.
Safety score: Rates websites based on SSL validity, domain age, and reputation.

How It Works
Monitors your browsing activity in the background.
Checks each visited site against phishing databases and suspicious patterns.
Alerts you with a clear warning if a threat is detected.
Blocks access to confirmed phishing sites.

Why Use It?
Phishing attacks are increasingly sophisticated. This extension helps you stay safe by providing real-time protection and educational feedback, so you can browse with confidence.

Installation

Clone the project
Add it to your browser (Chrome).

Browse safely!
