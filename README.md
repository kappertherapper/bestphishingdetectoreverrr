A lightweight browser extension that detects phishing websites in real-time.

<br>

✨ Features

- Analyzes URLs for suspicious patterns.
- Warns you immediately when visiting a potentially malicious site.

<br>

💡 Why Use It?
- Phishing attacks are becoming increasingly sophisticated. 
- This extension helps you stay safe so you can browse with confidence.

<br>

📥 Installation
Clone the project:

 git clone https://github.com/your-repo/bestphishingdetectoreverrr.git


- Add it to your browser (Chrome):
- Open Chrome and go to chrome://extensions/.
- Enable Developer mode (toggle in the top-right corner).
- Click Load unpacked and select the cloned project folder.

<br>

Browse safely! 🎉
